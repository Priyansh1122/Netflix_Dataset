# Netflix_Dataset

1-Project Overview:

This project focuses on analyzing the Netflix dataset to uncover insights about movies and TV shows, including trends, content distribution, ratings, and top contributors.

2-Objectives:
             Analyze Netflix content based on year, country, category, and ratings,
             Identify trends in Movies vs TV Shows,
             Find top directors and actors,
            Answer real-world business questions using data

3-Dataset:
         Source: Kaggle ,
        Contains information about Netflix titles such as show ID, title, director, cast, country, release year, rating, and duration

4-Tools & Technologies:
                       Python,
                       Pandas,
                       NumPy,
                       Seaborn & Matplotlib, 
                       Jupyter Notebook

5-Data Cleaning Steps:
                     Removed duplicate records,
                     Handled missing/null values,
                     Converted date columns to datetime format,
                     Filtered and structured data for analysis

6-Key Insights:
               Movies dominate Netflix content compared to TV Shows,
               Significant growth in content after 2015,
               Certain countries contribute the majority of content,
               Popular ratings include TV-14 and TV-MA
